# 🚗 Car Sales Dashboard

## Project Overview
This project analyzes car sales data using Power BI to provide insights into sales performance, revenue, and customer trends.

## Tools Used
- Power BI
- SQL
- Excel

## Features
- YTD Total Sales
- YTD Average Sales
- YTD Cars Sold
- Monthly Sales Analysis
- Dealer-wise Sales Analysis
- Company-wise Sales Analysis
- Interactive Filters

## Skills
- Data Cleaning
- Data Visualization
- Dashboard Development
- DAX
- Business Intelligence
